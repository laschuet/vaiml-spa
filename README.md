vaiml-spa.

## Requirements

* [Node.js](https://nodejs.org/en/)
* npm

## Usage

```
npm install
```
Install all dependencies.

```
npm run dev
```
Start the development server.

```
npm run dist
```
Build for production.

```
npm run lint
```
Lint the code.

```
npm run lint-fix
```
Lint the code including fixes.

```
npm run prettify
```
Run prettier.

```
npm run prettify-fix
```
Run prettier including fixes.
